# 面試準備 辛耕

## 要求

AI 模型整合：介接 LLM API，如 OpenAI、Claude，或部署地端模型，開發企業級 AI Chatbot 與 RAG 知識庫系統。

- 自動化流程開發：熟悉 Vibe Coding，能運用 n8n、Python 腳本或 RPA 工具，串接內部系統 ERP/CRM 與 AI 介面，實現業務流程自動化。
- 數據工程：處理異質資料清洗與向量資料庫維護，優化 AI 回應精準度。
- 技術評估：持續追蹤最新 AI 框架，評估並導入適合公司的技術方案。

## 必要條件

- 編程能力：精通 Python 等人工智慧開發常用程式語言，熟悉 API 串接與非同步處理。
- AI 實務：具備 Vibe Coding 經驗，有 RAG 架構或 Agent 開發實績。
- 資料庫：具備 SQL 使用經驗，並瞭解向量資料庫運作原理。

## 加分條件

- 具備大型語言模型地端部署 Local LLM 經驗。
- 熟悉前端框架，如 React、Vue，可獨立產出簡單原型。

## 面試準備方向

- 能說明 LLM API 串接流程：API key、request/response、prompt、system message、錯誤處理、串流輸出、成本與延遲。
- 能說明地端模型部署：Ollama、模型選擇、硬體限制、隱私優勢、速度與效果取捨。
- 能說明 RAG 流程：文件讀取、切 chunk、embedding、向量資料庫、retrieval、prompt 組裝、回答與引用來源。
- 能說明 Agent：工具呼叫、任務拆解、記憶、流程控制，以及和一般 chatbot 的差異。
- 能準備一個作品說法：本機論文 RAG 或 Notion 筆記助理。
