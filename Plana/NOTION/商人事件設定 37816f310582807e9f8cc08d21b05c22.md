## 商人定位

商人屬於特殊事件。

不會出現在一般樓層。

---

## 出現樓層

固定出現在：

```
第 3 層
第 5 層

第 7 層
第 10 層

第 13 層
第 15 層

第 17 層
第 20 層

第 23 層
```

即：

```
所有中 Boss 樓層
+
所有 Boss 樓層
```

---

## 生成方式

每個符合條件的樓層：

```
商人 × 1
```

---

## 生成規則

商人不額外增加事件數量。

而是：

```
替換一個空地事件
```

因此：

```
總事件數仍維持 25
```

---

## 商人提示

商人事件顯示：

```
似乎有影子在晃動
```

玩家無法事先分辨：

```
普通怪
精英怪
稀有怪
商人
```

---

## 商人位置

商人會被放入本層的 25 個事件池中。

玩家不一定會遇到商人。

是否能遇到取決於玩家的路線選擇。

# 裝備品質成長

商人販售的裝備會依樓層提升品質。

## 第 1 ~ 5 層

```

```


---

## 第 6 ~ 10 層

```

```

```
皮甲
鐵甲
```

---

## 第 11 ~ 15 層

止血繃帶

效果：
解除流血
恢復 10 HP

```
鐵劍
鋼劍
```

```
鐵甲
鋼甲
```
