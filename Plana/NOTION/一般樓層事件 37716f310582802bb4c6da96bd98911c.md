# 一般樓層事件配置

## 事件總數

每層固定生成：

```
25 個事件
```

計算方式：

```
5 步
×
5 個事件
=
25 個事件
```

---

## 怪物保底規則

每一步至少存在：

```
1 個普通怪物
```

確保玩家每一步都有戰鬥選項。

---

## 事件配置

### 普通怪

生成：

```
7
```

---

### 精英怪

生成：

```
1~2
```

精英怪為額外生成事件。

不替換普通怪。

---

### 金錢

生成：

```
6
```

---

### 道具

生成：

```
3
```

---

### 寶箱事件

生成：

```
3
```

開啟需要：

```
寶箱鑰匙 × 1
```

開啟後：

```
50%
寶箱

50%
寶箱怪
```

---

### 陷阱

生成：

```
2
```

觸發後進行逃脫判定。

失敗可能：

```
扣除 HP
扣除 Gold
失去道具
```

---

### 空地

生成剩餘數量補足。

目前配置下約：

```
2~3
```

---

## 空地效果

玩家可在空地休息。

效果：

```
恢復少量生命值
```

具體數值後續設計。

---

## 稀有怪

每層最多：

```
1 隻
```

---

### 金色稀有怪

生成機率：

```
1 / 100
```

---

### 銀色稀有怪

生成機率：

```
1 / 10
```

---

### 生成方式

若成功生成：

```
替換一個空地事件
```

不增加事件總數。

---

## 商人

一般樓層不生成商人。

商人固定出現在：

```
第 3 層
第 7 層
第 13 層
第 17 層
第 23 層
```

中 Boss 樓層。

位於中 Boss 戰前。
