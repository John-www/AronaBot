# 逃跑事件

# 逃跑系統

## 逃跑規則

除以下敵人外：

```
各區大Boss

最終Boss（魔王）
```

皆可嘗試逃跑。

---

# 戰鬥前逃跑

玩家遭遇怪物時，可選擇：

```
1. 戰鬥

2. 逃跑

3. 使用道具
```

---

## 逃跑成功率

| 敵人類型 | 成功率 |
| --- | --- |
| 普通怪 | 80% |
| 精英怪 | 50% |
| 中Boss | 20% |
| 大Boss | 0% |
| 魔王 | 0% |

---

## 逃跑成功

效果：

```
返回地圖

本次事件消失
```

---

## 逃跑失敗

效果：

```
怪物先攻一次

之後進入戰鬥
```

---

# 戰鬥中逃跑

戰鬥進行中可選擇：

```
1. 攻擊

2. 使用道具

3. 逃跑
```

---

## 逃跑成功率

| 敵人類型 | 成功率 |
| --- | --- |
| 普通怪 | 50% |
| 精英怪 | 25% |
| 中Boss | 10% |
| 大Boss | 0% |
| 魔王 | 0% |

---

## 逃跑成功

效果：

```
返回地圖

本次事件消失
```

---

## 逃跑失敗

效果：

```
怪物立即攻擊一次
```

---

# 煙霧彈

使用後：

```
立即成功逃跑
```

不需進行判定。

---

## 可使用對象

```
普通怪

精英怪

中Boss
```

---

## 無法使用對象

```
各區大Boss

最終Boss（魔王）
```

---

# 特殊規則

## 麻痺狀態

玩家處於：

```
麻痺
```

時：

```
無法逃跑
```

---

## 逃跑不消耗回合

逃跑判定屬於特殊行動。

成功：

```
直接離開戰鬥
```

失敗：

```
怪物立即反擊
```

---

# 設計理念

```
普通怪可自由選擇是否戰鬥

精英怪與中Boss可視情況撤退

大Boss與魔王必須正面迎戰

煙霧彈作為保命道具存在

避免玩家因運氣不佳被迫死亡
```
