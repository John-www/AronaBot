# Codex 筆記閱讀順序

## 1. 遊戲核心概念

先閱讀：

```
地城樓層設計
事件生成規則
步數系統
```

目的：

```
理解遊戲是25層地城

每層5步

每步提供5個選項

經過5個事件後進入樓梯或Boss事件
```

---

## 2. 玩家基礎系統

接著閱讀：

```
冒險者系統
背包系統
死亡系統
逃跑系統
通關事件
```

目的：

```
理解玩家HP、裝備、背包容量、死亡、復活、通關流程
```

---

## 3. 戰鬥系統

接著閱讀：

```
戰鬥流程
異常狀態總表
道具系統
```

目的：

```
理解攻擊、防禦、狀態異常、使用道具、回合流程
```

---

## 4. 裝備系統

接著閱讀：

```
武器系統
防具系統
飾品系統
裝備幸運詞綴系統
數值總表
```

目的：

```
理解裝備數值、Boss裝、寶箱裝、隨機詞綴
```

---

## 5. 事件系統

接著閱讀：

```
一般樓層事件
特殊樓層事件
金錢事件
寶箱系統
寶箱怪系統
商人系統
```

目的：

```
理解探索時可能遇到的事件與獎勵
```

---

## 6. 怪物與Boss

接著閱讀：

```
第一區怪物系統
第二區怪物系統
第三區怪物系統
第四區怪物系統
第五區怪物系統

各區中Boss
各區Boss
最終Boss
```

目的：

```
理解怪物池、出現權重、數值、掉落與Boss機制
```

---

## 7. AI旁白系統

最後閱讀：

```
AI旁白與GM對話系統
```

目的：

```
AI只負責文字表現

不負責遊戲判定

所有數值與結果由Bot決定
```

---

# 建議實作順序

## 第一階段：先做可玩的最小版本

```
/start
/player
/move
/fight
/item
/shop
```

先支援：

```
第1區 1~5層

普通怪

精英怪

第3層中Boss

第5層Boss

死亡

逃跑

寶箱

商人
```

---

## 第二階段：擴充完整25層

```
加入第2~5區怪物

加入所有Boss

加入裝備總表

加入完整道具
```

---

## 第三階段：加入進階系統

```
裝備幸運詞綴

寶箱怪

通關事件

死亡結算
```

---

## 第四階段：加入AI旁白

```
事件結果先由Bot計算

再交給AI生成GM文字

AI不可改變結果
```
