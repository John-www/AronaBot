# 事件系統

## 系統概述

每層固定由：

```
5 個步（Step）
+
1 個樓層結束事件
```

組成。

---

## 步（Step）

每一步提供：

```
5 個事件選項
```

玩家從中選擇 1 個事件。

未選擇的事件立即消失。

不可返回重新選擇。

---

## 步數規則

所有事件都會消耗一步。

包含：

- 普通怪
- 精英怪
- 稀有怪
- 金錢
- 道具
- 寶箱
- 寶箱怪
- 陷阱
- 空地
- 商人

流程：

```
選擇事件
↓
處理事件
↓
步數 +1
```

---

# 事件總覽

## 戰鬥事件

### 普通怪

最基本的戰鬥事件。

玩家可選擇：

```
戰鬥
逃跑
```

逃跑成功後怪物消失。

---

### 精英怪

強度高於普通怪。

低於中 Boss。

玩家可選擇：

```
戰鬥
逃跑
```

具體數值與掉落後續設計。

---

### 銀色稀有怪

特殊獎勵怪物。

主要掉落大量金錢。

不以攻擊玩家為主要目的。

經過一定回合後會主動逃跑。

玩家需在逃跑前擊敗牠。

---

### 金色稀有怪

極稀有獎勵怪物。

主要掉落大量金錢。

出現機率極低。

經過一定回合後會主動逃跑。

玩家需在逃跑前擊敗牠。

---

### 寶箱怪

由寶箱事件觸發。

玩家開啟寶箱後：

```
50%
寶箱怪
```

立即進入戰鬥。

不可逃跑。

---

# 資源事件

### 金錢

直接獲得 Gold。

數量依樓層與平衡設計決定。

---

### 道具

獲得隨機道具。

可能包含：

- 火把
- 地圖
- 回復藥
- 寶箱鑰匙

等物品。

詳細內容後續設計。

---

### 寶箱

需要：

```
寶箱鑰匙 × 1
```

才能開啟。

開啟後：

```
50%
寶箱

50%
寶箱怪
```

玩家無法事先分辨。

---

# 功能事件

### 商人

固定出現在：

```
第 3 層
第 7 層
第 13 層
第 17 層
第 23 層
```

即：

```
中 Boss 樓層
```

位於中 Boss 戰前。

玩家可使用 Gold 購買商品。

販售內容後續設計。

---

# 特殊事件

### 陷阱

觸發後進行逃脫判定。

---

### 成功

```
成功逃脫
↓
無事發生
```

---

### 失敗

可能發生：

```
扣除 HP
扣除 Gold
失去道具
```

具體數值後續設計。

---

### 空地

可作為休息地。

玩家可恢復少量生命值。

恢復數值後續設計。

---

# 事件提示系統

玩家不會直接看到事件名稱。

系統只提供簡單提示。

---

### 似乎有影子在晃動

可能為：

- 普通怪
- 精英怪
- 銀色稀有怪
- 金色稀有怪
- 商人

---

### 似乎有什麼閃閃發光

可能為：

- 金錢
- 寶箱

---

### 似乎是安全的地方

可能為：

- 道具
- 空地
- 陷阱

---

### ???

未公開事件。

無法得知任何資訊。

---

# 情報道具

### 火把

效果：

```
查看下一步全部 5 個事件
```

使用後直接顯示事件名稱。

---

### 地圖

效果：

```
查看本層全部 25 個事件
```

使用後揭露：

- 所有事件
- 所在步數
- 所在位置
