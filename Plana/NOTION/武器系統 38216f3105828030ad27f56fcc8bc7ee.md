# 武器系統

玩家可透過以下方式取得道具：

```
地城探索
商人購買
怪物掉落
寶箱獎勵
```

## 武器規則

武器決定玩家攻擊傷害。

每次攻擊時，會於武器傷害範圍內隨機取值。

例如：

```
木劍
1 ~ 5
```

攻擊時可能造成：

```
1
2
3
4
5
```

任一數值傷害。

# 特殊武器

特殊武器不會出現在一般商店。

取得方式：

```
中 Boss 掉落
Boss 掉落
寶箱獲得
```

---

# Boss武器

## 取得方式

### 中 Boss

掉落機率：

```
20%
```

---

### Boss

掉落機率：

```
100%
```

---

## 武器特色

提高傷害上限。

規則：

```
最低傷害不變
最高傷害提高
```

---

### 範例

```
銅劍
2 ~ 7
```

↓

```
哥布林酋長的棍棒
2 ~ 10
```

---

```
鐵劍
4 ~ 10
```

↓

```
骸骨騎士的大劍
4 ~ 13
```

---

## 命名規則

Boss武器名稱使用：

```
Boss名稱
+
專屬武器
```

例如：

```
哥布林酋長的棍棒
骸骨騎士的大劍
```

實際名稱於 Boss 系統設計時決定。

# 寶箱武器

## 取得方式

```
寶箱獎勵
```

---

## 武器特色

提高傷害下限。

規則：

```
提高最低傷害
最高傷害不變
```

---

### 範例

```
銅劍
2 ~ 7
```

↓

```
閃亮的銅劍
4 ~ 7
```

---

```
鐵劍
4 ~ 10
```

↓

```
閃亮的鐵劍
7 ~ 10
```

---

## 命名規則

寶箱武器名稱使用：

```
閃亮的
+
原武器名稱
```

例如：

```
閃亮的銅劍
閃亮的鐵劍
閃亮的鋼劍
```
