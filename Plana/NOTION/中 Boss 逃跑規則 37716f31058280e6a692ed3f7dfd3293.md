# 中 Boss 逃跑規則

## 條件

玩家必須持有：

```
寶箱鑰匙 × 1
```

才可選擇逃跑。

若沒有寶箱鑰匙：

```
不可選擇逃跑
必須進入戰鬥
```

---

## 逃跑流程

```
持有寶箱鑰匙
↓
進行逃跑判定
```

---

## 逃跑成功

```
逃跑判定成功
↓
消耗寶箱鑰匙 × 1
↓
略過中 Boss 戰鬥
↓
前往樓梯
↓
下一層
```

---

## 逃跑失敗

```
逃跑判定失敗
↓
不消耗寶箱鑰匙
↓
強制進入中 Boss 戰鬥
```

---

## 寶箱鑰匙用途

### 開啟寶箱

```
寶箱鑰匙 × 1
↓
開啟寶箱或寶箱怪
```

---

### 中 Boss 逃跑

```
持有寶箱鑰匙
↓
逃跑判定成功
↓
消耗寶箱鑰匙 × 1
↓
直接通過中 Boss 樓層
```

---

## 使用規則

寶箱鑰匙為消耗品。

```
成功使用後
↓
消失
```

若逃跑失敗：

```
不消耗
```
