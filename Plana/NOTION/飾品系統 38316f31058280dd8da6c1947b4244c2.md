# 飾品系統

## 飾品規則

飾品提供額外能力。

與武器、防具不同。

飾品主要影響：

```
戰鬥
探索
經濟
生存能力
```

---

## 飾品欄位

玩家僅有：

```
1 個飾品欄位
```

同時間只能裝備一件飾品。

更換裝備時：

```
原飾品卸下
↓
放回背包
```

---

# 飾品取得方式

## 怪物掉落

### 普通怪

```
不掉落飾品
```

---

### 精英怪

```
極低機率掉落飾品
```

---

### 中 Boss

```
有機率掉落飾品
```

主要來源之一。

---

### Boss

```
有機率掉落高品質飾品
```

---

## 商人販售

商人有機率販售飾品。

價格高於同階級武器、防具。

---

## 探索事件

探索獲得物品時：

```
金錢
>
道具
>
武器、防具
>
飾品
```

飾品為最稀有的探索獎勵之一。

---

# 飾品分類

## 基礎能力型（高出現率）

此類飾品最常出現。

會依樓層提升能力。

---

### 力量戒指

效果：

```
攻擊 +1
```

高階版本：

```
攻擊 +2
攻擊 +3
```

---

### 守護戒指

效果：

```
防禦 +1
```

高階版本：

```
防禦 +2
防禦 +3
```

---

### 生命護符

效果：

```
最大 HP +20
```

高階版本：

```
最大 HP +30
最大 HP +50
```

---

# 飾品定位

```

```

---

## 特殊功能型

特色：

```
策略性
功能性
```

# 高階飾品

高階飾品不會出現在一般商店。

取得方式：

```
Boss掉落
特殊寶箱
```

---

# 寶箱專屬飾品

寶箱專屬飾品為進階版本飾品。

# Boss專屬飾品

Boss專屬飾品為最稀有裝備之一。

通常擁有兩種效果。

---

## 類型一

基礎能力複合

例如：

```
攻擊 +2
防禦 +2
```

---

```
攻擊 +2
最大 HP +30
```

---

```
防禦 +2
最大 HP +30
```

---

## 類型二

基礎能力 + 狀態抗性

例如：

```
攻擊 +2
免疫中毒
```

---

```
防禦 +2
免疫燒傷
```

---

```
最大 HP +30
免疫麻痺
```

---

## 類型三（後期Boss）

以經濟與生存能力為主。

主要出現在：

```
第15層後
```

---

### 範例

```
擊敗怪物 Gold +50%
擊敗怪物恢復 5 HP
```

---

```
販售物品 Gold +50%
最大 HP +50
```

---

```
擊敗怪物 Gold +50%
攻擊 +3
```

---

```
擊敗怪物恢復 5 HP
防禦 +3
```

---

# 設計原則

普通飾品：

```
單一能力
```

---

寶箱專屬飾品：

```
強化版單一能力
```

---

Boss專屬飾品：

```
複合能力
```

作為後期主要畢業裝備來源。
