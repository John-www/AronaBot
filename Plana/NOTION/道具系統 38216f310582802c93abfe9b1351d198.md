# 道具系統

## 道具取得方式

玩家可透過以下方式取得道具：

```
地城探索
商人購買
怪物掉落
寶箱獎勵
```

---

# 消耗道具

## 回復藥

效果：

```
恢復 20 HP
```

使用時機：

```
戰鬥中可使用
戰鬥外可使用
```

使用後消耗。

---

## 高級回復藥

效果：

```
恢復 50 HP
```

使用時機：

```
戰鬥中可使用
戰鬥外可使用
```

使用後消耗。

---

## 解毒藥

效果：

```
解除所有異常狀態
```

包含：

```
中毒
燒傷
麻痺
```

使用後消耗。

---

## 火把

效果：

```
查看下一步全部 5 個事件
```

使用後直接顯示：

```
普通怪
精英怪
商人
金錢
寶箱
```

等真實事件名稱。

使用後消耗。

---

## 地圖

效果：

```
查看本層全部事件
```

揭露：

```
全部 25 個事件
以及所在步數與位置
```

使用後消耗。

---

## 煙霧彈

效果：

```
本次逃跑必定成功
```

適用：

```
普通怪
精英怪
```

不可使用於：

```
中 Boss
Boss
最終魔王
寶箱怪
```

使用後消耗。

---

# 特殊道具

## 寶箱鑰匙

效果：

```
開啟寶箱
```

消耗：

```
1 把
```

開啟後：

```
50%
寶箱

50%
寶箱怪
```

---

# 狀態異常

## 中毒

效果：

```
每回合扣除 3 HP
```

持續：

```
直到解除
```

解除方式：

```
解毒藥
```

---

## 燒傷

效果：

```
每回合扣除 5 HP
```

持續：

```
3 回合
```

解除方式：

```
解毒藥
```

---

## 麻痺

效果：

```
無法進行攻擊
```

仍可：

```
使用道具
嘗試逃跑
```

持續：

```
1 回合
```

解除方式：

```
解毒藥
```

---

# 戰鬥中使用道具

玩家回合可選擇：

```
攻擊
使用道具
逃跑
```

---

使用道具後：

```
消耗本回合行動
```

例如：

```
玩家使用回復藥
↓
恢復 20 HP
↓
怪物行動
```

---

# 道具販售

玩家可將背包內道具販售給商人。

---

```
解毒藥
火把
地圖
煙霧彈
寶箱鑰匙
```

販售價格：

```
約為原價 50%
```
