# 通關事件

## 觸發條件

擊敗：

```
第25層最終Boss
魔王
```

---

觸發流程：

```
擊敗魔王

↓

獲得「勇者之證」

↓

進入通關事件

↓

顯示冒險結果

↓

恭喜通關

↓

遊戲結束
```

---

# 通關獎勵

## 勇者之證

類型：

```
特殊紀念物品
```

用途：

```
代表成功通關地城
```

特性：

```
不可販售

不可丟棄

永久保留
```

---

# 冒險結果

## 通關時間

台灣時間 如 2026/06/21/ 13:10

---

## 擊敗怪物數量

顯示：

```
普通怪擊敗數

精英怪擊敗數

Boss擊敗數

總擊敗數
```

範例：

```
普通怪：85

精英怪：18

Boss：9

總計：112
```

# 通關訊息

```
恭喜通關！

你成功擊敗了魔王，
取得了勇者之證。

你的冒險將被永遠記錄於地城之中。
```

---

## 最終生命值

顯示：

```
通關時剩餘生命值
```

範例：

```
37 / 280 HP
```

或

```
245 / 280 HP
```

---

用途：

```
記錄通關時的生存狀態

區分驚險通關與輕鬆通關

增加紀念價值
```

# 設計理念

```
讓玩家回顧整場冒險

展示本次Build成果

增加通關成就感

方便未來新增排行榜或紀錄系統
```
