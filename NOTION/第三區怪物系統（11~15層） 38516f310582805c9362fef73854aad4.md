# 第三區設計理念

怪物分工：

```
狼         → 流血

毒蛇       → 劇毒

病鼠       → 疾病

猴子       → 先攻＋閃避

超甲狂犀   → 蓄力爆發

吸血蝠     → 吸血＋中毒

獅子       → 純數值
```

---

第三區特色：

```
開始出現全新異常狀態

怪物機制複雜度提升

玩家開始需要考慮狀態管理

開始出現閃避與蓄力型敵人
```
