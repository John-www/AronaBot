# Discord地城遊戲基本規格書

## 遊戲類型

Discord文字地城冒險遊戲。

玩家透過Discord指令或按鈕進行探索、戰鬥、購買道具、使用道具與通關。

---

# 遊戲目標

玩家從第1層開始探索地城。

最終目標：

```
抵達第25層

擊敗魔王

獲得勇者之證

進入通關事件
```

---

# 地城結構

## 總樓層

```
25層
```

---

## 區域劃分

```
1~5層：哥布林區

6~10層：亡靈墓穴

11~15層：野獸荒原

16~20層：詛咒森林

21~25層：深淵魔域
```

---

## 每層流程

每層有：

```
5步
```

每一步提供：

```
5個事件選項
```

玩家選擇其中一個事件。

經過5個事件後：

```
一般樓層 → 樓梯事件

中Boss樓層 → 中Boss事件

大Boss樓層 → Boss事件

第25層 → 魔王事件
```

# 初始玩家狀態

```
HP：100 / 100

Gold：0

武器：木劍 1~5

防具：布衣 +1

飾品：無

背包容量：15
```

---

# 戰鬥流程

基本流程：

```
玩家先攻

↓

怪物反擊

↓

下一回合
```

部分怪物具有：

```
先制攻擊

二連擊

三連擊

異常攻擊

吸血

回血

蓄力
```

---

# 傷害公式

```
玩家造成傷害 = 武器隨機傷害 + 攻擊加成
```

```
玩家受到傷害 = 怪物攻擊 - 玩家防禦
```

最低傷害：

```
1
```

---

# 玩家行動

戰鬥中玩家可選：

```
1. 攻擊

2. 使用道具

3. 逃跑
```

---

# 逃跑規則

可逃跑：

```
普通怪

精英怪

中Boss
```

不可逃跑：

```
大Boss

魔王
```

煙霧彈可直接逃離普通怪、精英怪、中Boss。

---

# 死亡規則

當玩家：

```
HP ≤ 0
```

先檢查是否持有：

```
鳳凰羽毛
```

若有：

```
消耗1個鳳凰羽毛

恢復30HP

繼續遊戲
```

若沒有：

```
顯示 YOU DIED

顯示死亡結算

返回主選單
```

---

# 背包規則

背包容量：

```
初始：15格

中背包：20格

大背包：30格
```

所有道具與備用裝備占用背包。

已裝備中的武器、防具、飾品不占背包。

Gold不占背包。

```
回復藥

大回復藥

解毒藥

止血繃帶

萬靈藥

火把

地圖

```

---

# 裝備系統

裝備分為：

```
武器

防具

飾品
```

玩家同時可裝備：

```
1把武器

1件防具

1件飾品
```

---

# 武器

武器提供攻擊傷害範圍。

例如：

```
木劍：1~5

銅劍：2~7

鐵劍：4~10
```

---

# 防具

防具提供固定防禦力。

例如：

```
布衣：+1

皮甲：+2

鐵甲：+4
```

---

# 飾品

飾品提供特殊效果。

例如：

```
攻擊加成

防禦加成

最大HP增加

異常免疫

Gold增加

吸血

事件公開增加
```

---

# 裝備幸運詞綴

普通武器與防具有30%機率出現幸運詞綴。

寶箱裝備必定有幸運詞綴。

Boss裝備不套用隨機詞綴。

---

事件類型包含：

```
普通怪

精英怪

金錢

道具

寶箱

寶箱怪

陷阱

空地

商人

稀有怪
```

---

# 寶箱系統

開寶箱需要：

```
寶箱鑰匙 ×1
```

開啟後：

```
90% 普通寶箱

10% 寶箱怪
```

寶箱可獲得：

```
道具

Gold

武器

防具

飾品

特殊物品
```

---

# 商人系統

商人每次出現重新生成商品。

商品總數：

```
15項
```

保底商品：

```
當區武器 ×1

當區防具 ×1

回復藥

解毒藥

火把

寶箱鑰匙
```

第15層後新增：

```
止血繃帶

萬靈藥
```

---

# 通關事件

擊敗魔王後：

```
獲得勇者之證

顯示通關結算

遊戲結束
```

通關結算包含：

```
結束日期

擊敗怪物數

獲得Gold

使用道具數

最終生命值

最終裝備

背包剩餘物品
```

---

# AI GM系統

AI只負責：

```
旁白

氣氛描述

地城內建議
```

AI不可負責：

```
傷害計算

掉落判定

逃跑成功率

事件生成

道具消耗
```

所有遊戲判定由Bot程式處理。
