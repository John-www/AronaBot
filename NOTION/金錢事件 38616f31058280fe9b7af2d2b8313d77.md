# 金錢事件系統

## 金錢事件

玩家探索地城時有機會發現金錢。

代表：

```
遺失的錢袋

冒險者遺物

隱藏金庫

散落的金幣

怪物藏匿的財寶
```

---

# 金錢事件類型

## 一般金錢事件

機率：

```
95%
```

---

獲得 Gold：

| 樓層 | Gold |
| --- | --- |
| 1~5 | 10 ~ 50 |
| 6~10 | 30 ~ 100 |
| 11~15 | 80 ~ 200 |
| 16~20 | 150 ~ 400 |
| 21~25 | 300 ~ 800 |

---

事件範例：

```
你在角落發現一個破舊錢袋。

獲得 42 Gold。
```

---

## 大金錢事件

機率：

```
5%
```

---

獲得大量 Gold：

| 樓層 | Gold |
| --- | --- |
