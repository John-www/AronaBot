# 商人系統

## 商人事件

玩家探索地城時有機會遇到商人。

商人可提供：

```
武器

防具

飾品

消耗道具

背包升級
```

---

# 商人生成規則

每次遇到商人時：

```
重新生成商品
```

---

離開商店後：

```
無法返回
```

---

商品總數：

```
15項商品
```

---

# 常駐商品

20格

# 裝備生成規則

商人販售的裝備依樓層提升。

---

範例：

1~5層：

```
木棒

短刀

銅劍

皮甲
```

---

11~15層：

```
鋼劍

獵刀

雙刃斧

獸骨甲
```

---

21~25層：

```
魔鋼劍

黑曜石大劍

深淵重甲
```

---

# 飾品生成規則

商人可販售：

```
一般飾品

能力型飾品

抗性型飾品

功能型飾品
```

---

Boss飾品：

```
不會販售
```

---

# 玩家販售系統

商人可收購物品。

---

## 消耗品

售價：

```
原價 50%
```

---

範例：

```
回復藥

售價 50 Gold

↓

販售 25 Gold
```

---

## 裝備

售價：

```
原價 70%
```

---

範例：

```
鋼劍

售價 500 Gold

↓

販售 350 Gold
```

---

## 特殊物品

```
不可販售
```

---

包含：

```
勇者之證

鳳凰羽毛
```

---
