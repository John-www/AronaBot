# 防具系統

## 防具規則

防具提供固定防禦力。

受到傷害時：

```
最終傷害
=
怪物攻擊
-
玩家防禦
```

最低造成：

```
1 點傷害
```

---

# 一般防具

玩家可透過以下方式取得道具：

```
地城探索
商人購買
怪物掉落
寶箱獎勵
```

## 初始防具

### 布衣

取得方式：

```
初始裝備
```

防禦：

```
+1
```

---

# 特殊防具

特殊防具不會出現在一般商店。

取得方式：

```
中 Boss 掉落
Boss 掉落
寶箱獲得
```

---

# Boss防具

## 取得方式

### 中 Boss

掉落機率：

```
20%
```

---

### Boss

掉落機率：

```
100%
```

---

## 防具特色

提高防禦力。

規則：

```
當前階級防具
+
3~6 防禦
```

---

### 範例

```
皮甲
+2
```

↓

```
哥布林酋長的皮甲
+6
```

---

```
鐵甲
+4
```

↓

```
黑騎士的鎧甲
+9
```

---

## 命名規則

Boss防具名稱使用：

```
Boss名稱
+
專屬防具
```

例如：

```
哥布林酋長的皮甲
黑騎士的鎧甲
```

實際名稱於 Boss 系統設計時決定。

---

# 寶箱防具

## 取得方式

```
寶箱獎勵
```

---

## 防具特色

提高防禦力。

規則：

```
當前階級防具
+
1~3 防禦
```

---

### 範例

```
皮甲
+2
```

↓

```
堅固的皮甲
+3
```

---

```
鐵甲
+4
```

↓

```
堅固的鐵甲
+6
```

---

## 命名規則

寶箱防具名稱使用：

```
堅固的
+
原防具名稱
```

例如：

```
堅固的皮甲
堅固的鐵甲
堅固的鋼甲
```
