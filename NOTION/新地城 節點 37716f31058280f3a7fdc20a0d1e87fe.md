# DC 地城 - 遊戲流程概述

## 遊戲目標

玩家扮演冒險者探索 25 層地城。

在資源有限的情況下：

- 擊敗怪物
- 取得金錢與裝備
- 購買道具
- 挑戰 Boss

最終擊敗第 25 層的最終魔王並通關。

---

# 基本流程

## 開始冒險

玩家進入第 1 層地城。

每層固定包含：

```
5 個探索事件
+
1 個樓層結束事件
```

---

## 探索

每一步會出現：

```
5 個事件選項
```

例如：

```
1. 普通怪
2. 商人
3. 金錢
4. 寶箱
5. 道具
```

玩家選擇其中一個。

未選擇的事件會消失。

## 戰鬥

遭遇怪物後：

玩家可選擇：

```
戰鬥
或
逃跑
```

## 商人

商人不一定出現。

可販售：

- 寶箱鑰匙
- 道具
- 裝備

（詳細內容後續設計）

---

## 寶箱

開啟需要：

```
寶箱鑰匙 × 1
```

寶箱可能是：

```
寶箱
或
寶箱怪
```

玩家無法事先分辨。

# 冒險結束

以下情況結束本次冒險：

### 死亡

```
HP 歸零
↓
冒險失敗
```

---

### 通關

```
擊敗第 25 層最終魔王
↓
結算畫面
↓
通關
```
