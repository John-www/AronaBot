# 背包

# 背包系統

## 背包容量

### 小型背包（初始）

容量：

```
15 格
```

---

### 中型背包

容量：

```
20 格
```

---

### 大型背包

容量：

```
30 格
```

---

# 容量計算規則

所有持有物品皆占用背包容量。

每件物品占用：

```
1 格
```

---

## 範例

```
回復藥 × 5
=
5 格
```

```
解毒劑 × 3
=
3 格
```

```
寶箱鑰匙 × 2
=
2 格
```

```
火把 × 1
=
1 格
```

```
地圖 × 1
=
1 格
```

---

# 占用背包容量的物品

## 消耗品

- 回復藥
- 解毒劑
- 火把
- 地圖
- 其他消耗品

---

## 特殊道具

- 寶箱鑰匙

---

## 備用裝備

未裝備中的：

- 武器
- 防具
- 飾品

皆占用背包空間。

---

# 不占用背包容量的物品

## 已裝備物品

目前裝備中的：

```
武器
防具
飾品
```

不占用背包容量。

---

## Gold

```
Gold
```

不占用背包容量。

---

# 道具堆疊

相同道具可堆疊。

例如：

```
回復藥 × 10
```

顯示於同一格物品欄位。

但仍占用：

```
10 格容量
```

---

# 背包已滿

當背包容量達到上限時：

```
15 / 15
20 / 20
30 / 30
```

無法獲得新物品。

---

## 處理方式

玩家可選擇：

```
丟棄現有物品
```

或：

```
放棄本次獲得
```

---

# 顯示方式

範例：

```
HP：87 / 100
Gold：125

武器：木劍（1~5）
防具：布衣（防禦+1）
飾品：無

背包：11 / 15

回復藥 × 5
解毒劑 × 2
寶箱鑰匙 × 1
火把 × 1
地圖 × 1
```
