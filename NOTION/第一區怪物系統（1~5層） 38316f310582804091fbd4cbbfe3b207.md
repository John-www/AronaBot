# 第一區設計理念

怪物分工：

```
史萊姆     → 教學怪

哥布林     → 高血量

骷髏兵     → 高攻擊

殭屍       → 麻痺

鬼火       → 燒傷

毒蜘蛛     → 中毒

盜賊       → 先制攻擊
```

---

精英怪為普通怪的強化版本：

```
哥布林 → 哥布林隊長

毒蜘蛛 → 巨型蜘蛛

鬼火 → 大鬼火

殭屍 → 屍群
```

後續第二區怪物可由第一區精英怪降級為普通怪進行延伸。
