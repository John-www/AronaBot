# 武器&防具補充

# 裝備幸運詞綴系統

## 系統概念

部分武器與防具掉落時，

有機會附帶額外效果。

---

目的：

```
增加刷寶樂趣

提升裝備多樣性

讓同一件裝備有不同玩法
```

---

# 武器幸運詞綴

## 掉落機率

普通武器：

```
70% 普通武器

30% 幸運武器
```

---

Boss武器：

```
不適用
```

---

Boss武器維持固定效果。

---

## 顯示方式

範例：

```
獵刀

↓

獵刀（流血）

↓

獵刀（吸血）
```

---

# 寶箱裝備規則

## 寶箱武器

必定擁有：

```
閃亮效果
```

以及：

```
1個武器幸運詞綴
```

---

範例：

```
閃亮的鋼劍（吸血）

閃亮的戰矛（疾病）

閃亮的魔鋼劍（魅惑）
```

---

## 寶箱防具

必定擁有：

```
堅固效果
```

以及：

```
1個防具幸運詞綴
```

---

範例：

```
堅固的鋼甲（生命）

堅固的皮甲（抗毒）

堅固的銀甲（裝甲）
```

---

# 設計理念

```
普通裝備也有使用價值

增加裝備選擇

避免玩家只追求Boss裝

讓寶箱裝成為高價值裝備來源
```
