# Boss 事件出現規則

## 固定事件順序

所有樓層皆遵循固定流程：

```
第 1 步
↓
第 2 步
↓
第 3 步
↓
第 4 步
↓
第 5 步
↓
樓層結束事件
```

---

## 一般樓層

樓層結束事件：

```
樓梯
```

流程：

```
5 個事件
↓
樓梯
↓
下一層
```

---

## 中 Boss 樓層

樓層結束事件：

```
中 Boss
```

流程：

```
5 個事件
↓
中 Boss
↓
樓梯
↓
下一層
```

中 Boss 不會出現在前 5 步。

固定於樓層結束事件出現。

---

## Boss 樓層

樓層結束事件：

```
Boss
```

流程：

```
5 個事件
↓
Boss
↓
樓梯
↓
下一層
```

Boss 不會出現在前 5 步。

固定於樓層結束事件出現。

---

## 最終樓層

樓層結束事件：

```
最終魔王
```

流程：

```
5 個事件
↓
最終魔王
↓
結算畫面
```

最終魔王不會出現在前 5 步。

固定於樓層結束事件出現。
