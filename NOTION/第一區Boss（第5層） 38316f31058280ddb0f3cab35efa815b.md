# 第一區Boss（第5層）

## 哥布林王

定位：

```
第一區最終Boss
```

---

HP：

```
50
```

---

攻擊：

```
 6~ 10

```

## 第一階段

觸發條件：

```
HP > 25
```

能力：

```
普通攻擊
```

---

## 第二階段

觸發條件：

```
HP ≤ 25
```

哥布林王進入狂暴狀態。

新增能力：

```
15% 麻痺攻擊
```

效果：

```
玩家麻痺 1 回合
```

---

## 狂暴機制

觸發條件：

```
HP ≤ 10
```

效果：

```
下一次行動必定二連擊
```

---

範例：

```
哥布林王 HP 9

↓

觸發狂暴

↓

下一回合

攻擊
+
攻擊
```

---

## 設計定位

第一階段：

```
高傷害
```

---

第二階段：

```
高傷害
+
麻痺
```

---

瀕死階段：

```
高傷害
+
麻痺
+
二連擊
```

---

這樣玩家打王時會有很明顯的節奏：

```
50~26
正常戰鬥

↓

25以下
開始出麻痺

↓

10以下
開始拚命
```

而且不需要新增複雜技能，Codex 也很好實作：

```
ifhp<=10:
next_attack_double=True
elifhp<=25:
can_use_paralyze=True
```

這種邏輯非常直觀。
