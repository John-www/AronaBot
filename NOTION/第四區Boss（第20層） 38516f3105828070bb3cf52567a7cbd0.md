# 第四區Boss（第20層）

## 森林樹王

定位：

第四區最終Boss

---

# Boss戰規則

森林樹王不會立即出現。

玩家必須依序擊敗：

```
普通怪 ×3

↓

精英怪 ×1

↓

森林樹王
```

---

戰鬥期間：

```
生命值保留

異常狀態保留

裝甲保留

道具可正常使用
```

---

設計理念：

```
利用大量怪物消耗玩家資源

最後再與Boss決戰
```

---

# 專屬防具

## 黑木鎧

類型：

Boss防具

---

```

```

---

特殊能力

### 鞏固

每場戰鬥開始時：

```
獲得 30 點裝甲
```

---

裝甲規則：

```
普通攻擊優先扣除裝甲

裝甲歸零後

才開始扣除生命值
```

---

裝甲不會阻擋：

```
流血

中毒

劇毒

燒傷

麻痺

疾病
```

---

每場戰鬥結束後：

```
恢復 15 點裝甲
```

最高不超過：

```
30 點裝甲
```

---

定位：

```
續戰型防具
```
