```
哥布林戰士     → 平衡型

巨型蜘蛛       → 中毒

大鬼火         → 燒傷

屍群           → 麻痺

骷髏弓箭手     → 先制攻擊

幽魂           → 複合異常

食屍鬼         → 吸血
```

---

精英怪分工：

```
骸骨騎士       → 高攻擊

死靈法師       → 異常狀態

怨靈           → 爆發傷害

屍體奇美拉     → 全能型
```
