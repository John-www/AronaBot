## 寶箱事件

玩家探索地城時有機會發現寶箱。

---

# 開啟條件

開啟寶箱需要：

```
寶箱鑰匙 ×1
```

---

開啟後：

```
消耗 1 把寶箱鑰匙
```

---

# 寶箱怪判定

開啟寶箱時：

```
90% 普通寶箱

10% 寶箱怪
```

---

若觸發寶箱怪：

```
寶箱變成怪物

立即進入戰鬥
```

---

# 寶箱掉落機率

| 類型 | 機率 |
| --- | --- |
| 道具 | 25% |
| 金錢 | 20% |
| 武器 | 25% |
| 防具 | 15% |
| 飾品 | 10% |
| 特殊物品 | 5% |

---

# 道具池

可能獲得：

```
回復藥

大回復藥

解毒藥

止血繃帶

萬靈藥

火把

地圖

煙霧彈

寶箱鑰匙
```

---

# 武器掉落

可能獲得：

```
當前樓層對應武器

閃亮的武器
```

---

## 閃亮的武器

特色：

```
提高最低傷害
```

範例：

```
銅劍

2~7

↓

閃亮的銅劍

4~7
```

---

# 防具掉落

可能獲得：

```
當前樓層對應防具

堅固的防具
```

---

## 堅固的防具

特色：

```
額外增加防禦
```

數值：

```
+1 ~ +3 防禦
```

範例：

```
皮甲 +2

↓

堅固的皮甲 +4
```

---

# 飾品掉落

可能獲得：

```
一般飾品

區域飾品
```

---
