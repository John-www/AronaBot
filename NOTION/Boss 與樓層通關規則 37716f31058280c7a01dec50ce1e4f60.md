## 流程

```
5 個事件
↓
中 Boss
↓
樓梯
↓
下一層
```

---

## 通關方式

玩家遭遇中 Boss 後有兩種選擇：

### 方案一：擊敗中 Boss

```
遭遇中 Boss
↓
進入戰鬥
↓
擊敗中 Boss
↓
前往樓梯
↓
下一層
```

---

### 方案二：逃離中 Boss

玩家可選擇逃跑。

但必須持有：

```
寶箱鑰匙 × 1
```

才可進行逃跑判定。

---

## 中 Boss 逃跑規則

### 逃跑流程

```
持有寶箱鑰匙
↓
進行逃跑判定
```

---

### 成功

```
逃跑判定成功
↓
消耗寶箱鑰匙 × 1
↓
略過中 Boss 戰鬥
↓
前往樓梯
↓
下一層
```

---

### 失敗

```
逃跑判定失敗
↓
不消耗寶箱鑰匙
↓
強制進入中 Boss 戰鬥
```

---

### 無寶箱鑰匙

```
不可選擇逃跑
↓
必須進入戰鬥
```

---

# 寶箱鑰匙用途

## 開啟寶箱

```
寶箱鑰匙 × 1
↓
開啟寶箱或寶箱怪
```

---

## 中 Boss 逃跑

```
持有寶箱鑰匙
↓
逃跑判定成功
↓
消耗寶箱鑰匙 × 1
↓
直接通過中 Boss 樓層
```

---

## 使用規則

寶箱鑰匙為消耗品。

```
成功使用
↓
消失
```

若中 Boss 逃跑失敗：

```
不消耗
```

# Boss 樓層

## 出現樓層

```
第 5 層

第 15 層
第 20 層
```

---

## 流程

```
5 個事件
↓
Boss
↓
樓梯
↓
下一層
```

---

## Boss 規則

Boss 為強制戰鬥。

不允許：

```
逃跑
離開遭遇
跳過 Boss
使用寶箱鑰匙通關
```

---

## 通關方式

唯一方式：

```
擊敗 Boss
↓
前往樓梯
↓
下一層
```
