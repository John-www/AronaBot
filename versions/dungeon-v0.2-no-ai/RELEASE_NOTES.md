# Dungeon Bot v0.2-no-ai

Date: 2026-06-23

This snapshot preserves the current playable Discord dungeon version before AI GM narration is added.

Included:
- Shared server-wide dungeon save flow
- Slash commands and button-based dungeon controls
- Floors 1-5 first-area gameplay
- Exploration, encounters, battles, chest flow, shop, inventory, equipment, status effects
- SQLite-backed player state
- No AI GM narration or GM chat integration yet

Primary files:
- `discord_rag_bot.py`
- `dungeon_game.py`
- `dungeon_db.py`
- `requirements.txt`
